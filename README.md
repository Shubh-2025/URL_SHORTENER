## 📖 About

**Shortly** is a clean, blazing-fast URL shortener that transforms long, unwieldy links into sleek short URLs — no sign-up required. Built for modern teams and creators, it offers secure redirects and real-time link generation with minimal friction.

## ✨ Features

| Feature | Description |
|---|---|
| 🔗 **Instant Shortening** | Paste any long URL — smart parser handles protocol adjustments automatically |
| 📋 **One-Click Copy** | Instantly copies the short link to your clipboard for easy sharing |
| 📊 **Smart Analytics** | View click volumes, referrer insights, and geographic trends |
| 🔒 **Secure by Design** | Malware scanning, encrypted redirection, and privacy-first approach |
| ⚡ **Edge Network** | Sub-15ms latency worldwide with automatic failover |
| ✏️ **Custom Slugs** | Branded short links (e.g., `snip.io/your-brand`) available |
| 🌍 **No Signup Required** | Start shortening links immediately — zero friction |

## ⚙️ How It Works

```
01  PASTE & OPTIMIZE
    Insert any long link — the smart parser handles protocol adjustments.
        ↓
02  GENERATE SHORT LINK
    One click creates a unique, compact URL ready for any audience.
        ↓
03  COPY & SHARE
    Instant copy to clipboard; share on social media, emails, or docs.
```

## 🏗️ Installation & Local Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Shubh-2025/URL_SHORTENER
   ```

2. **Navigate to the project folder**
   ```bash
   cd URL_SHORTENER
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the server**
   ```bash
   npm start
   ```

5. **Open in your browser**
   ```
   http://localhost:<your-port>
   ```
