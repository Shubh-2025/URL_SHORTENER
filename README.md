# URL Shortener

A simple web application to shorten long URLs into compact, shareable links.

## Features
- Generate short URLs from long ones
- Redirect short URLs to original destinations
- Clean, responsive interface

## Installation
1. Clone the repository: `git clone https://github.com/Shubh-2025/URL_SHORTENER`
2. Navigate to the project folder: `cd URL_SHORTENER`
3. Install dependencies: `npm install` (assuming Node.js)
4. Start the server: `npm start`

## Usage
- Access the live demo at: [https://url-shortener-xiao.onrender.com/](https://url-shortener-xiao.onrender.com/)
- Enter a long URL to generate a short version.
